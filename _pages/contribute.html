Contributing

Is there a library or module that we missed? Is there a library you think would be useful? Let us know using the form below!

The only requirements we have are that the library be related in some way to the omics (proteomics, lipidomics, metabolomics, genomics, etc.) and that it be documented. "Looking at the code" or "running it for a list of options" do not count as documentation! The library must have a proper README or documentation with a description of all methods and preferably an example for each method.
